# 演習 04: HTTPリクエストでAPI連携

## シナリオ
Contoso のサポートエンジニアは、社内システムにない最新情報（株価や外部サービスの指標など）を顧客に提供する必要があります。Copilot エージェントが外部 REST API を直接呼び出し、チャットで結果を返す仕組みを構築します。

## 目標
- HTTPリクエストノードの基本理解
- 外部APIからデータ取得し、会話内で返す方法を学ぶ
