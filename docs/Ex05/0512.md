# タスク 12: ナレッジソースの削除

## はじめに

このタスクでは、Copilot Studio でナレッジソースの削除方法を学びます。

## 説明

不要になったナレッジソースを削除します。

## 成功基準

- ナレッジソースを正しく削除できた。

## 主な作業

### 01: 削除手順

<details markdown="block">
  <summary><strong>解答を表示するにはこのセクションを展開してください</strong></summary>

1. ナレッジソース管理画面を開きます。
1. 削除したいナレッジソースを選択し、削除します。

</details>
