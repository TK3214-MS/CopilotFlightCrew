# タスク 4: Webサイトナレッジソースの設定

## はじめに
Contoso の顧客に追加情報を提供するため、Webサイトをナレッジソースとして統合します。

## 説明
このタスクでは、WebサイトのURLやアクセス権限を設定し、エージェントがWebサイトから情報を取得できるようにします。

## 成功基準
- 正しいURLとアクセス権限でWebサイトナレッジソースを設定できた

## 主な作業
1. Webサイトナレッジソースの設定
2. サイトURL・説明の入力
3. アクセス権限の設定
