# タスク 06: 条件分岐で変数を活用

## はじめに
顧客入力に応じて最適な応答を行うため、エージェントは保存済みデータをもとに条件分岐を行う必要があります。

## 説明
このタスクでは、変数の値に基づいて会話トピック内で条件分岐を設定し、ユーザー入力や収集済み情報に応じて会話の流れを制御します。

## 成功基準
- 変数値に基づく条件ロジックを作成できた
- 変数による会話分岐が正しく動作することをテスト・確認できた
