# タスク 04: Copilot の公開とテスト

## はじめに

このタスクでは、作成した Copilot を公開し、テストする方法を学びます。

## 説明

Copilot Studio で Copilot を公開し、テストチャットで動作を確認します。

## 成功基準

- Copilot を正常に公開できた。
- テストチャットで動作を確認できた。
